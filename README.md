# MEAN Stack Angular 16 client template
This is a simple way to get started in a mean project.

This project includes:
- Angular 16
- Taiwindcss
- Chart.js
- Material icons
- Bootstrap icons

# Views

- `/` Portal boilerplate
- `/auth/login` Login
- `/auth/signup` Sign Up
- `/dashboard` Admin Dashboard
